Mastermind

How to Play:

  • Guess the hidden 4-color code. You have 10 attempts.
  • You can interact with pins in 3 ways:
    • Drag colored pins from the bottom palette to any slot
    • Click on a color to automatically place it in the next empty slot
    • Click on a placed pin to remove it
  • Click "Check" to submit your guess
  • For each pin you place, you'll receive feedback:
    • Green: This pin has the correct color in the correct position
    • Orange: This pin has a color that exists in the secret code but in a different position
    • Black: This pin has a color that does not exist in the secret code
